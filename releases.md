# Release v1.4.4: The Timeline Cleanup Update

This update brings a highly requested timeline management tool and several under-the-hood polish improvements to make LiveUtilities feel even more native to Fusion 360! Perfect for tidying up right before a live class. 🚀

### ✨ What's New
* **Regroup Features:** Does your timeline have too many individual `CFG_Joint_XXX` groups? You can now select a contiguous block of grouped features in your timeline and instantly merge them into a single, consolidated `CFG_` group right from the **Live Config** tab.
* **Silent Add-in Detection:** Upgraded the background listener to automatically detect timeline changes generated by silent or API-heavy third-party plugins (like *Finger Joints Live*). LiveUtilities will now auto-refresh seamlessly without needing a manual rescan.
* **Custom UI Modals:** Say goodbye to those ugly, generic browser `confirm()` popups! All destructive actions (like deleting parameters or snapshots) now use clean, custom HTML modals that perfectly inherit your active theme's colors and fonts.
* **UI Polish:** Added primary-color hover effects to snapshot buttons and polished the visual hierarchy of the Config tab for a sleeker experience.

### 🔧 Under the Hood
* Bypassed a notorious Fusion API bug where clicking Timeline Groups obscures their underlying API entities, ensuring rock-solid selection detection.
* Restructured group-merging logic to directly target backend timeline indices, completely eliminating failure points caused by Fusion's string-sanitization rules during group renaming.

### 📦 Installation
Download the `Source code (zip)` below. Extract the folder, rename it to `LiveUtilities`, and place it in your Fusion Scripts directory:
* **Windows:** `%appdata%\Autodesk\Autodesk Fusion 360\API\Addins`
* **Mac:** `~/Library/Application Support/Autodesk/Autodesk Fusion 360/API/Addins`
